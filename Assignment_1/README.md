# Java Assignment I – Basic Programming & Number Problems

MCA Object Oriented Programming with Java Assignment

This assignment contains beginner to intermediate Java programs focused on loops, number operations, mathematical logic, sequences, and problem-solving fundamentals.

## Topics Covered

1. Fibonacci Sequence and Sum Calculation  
2. Prime Numbers from 1–1000  
3. Fibonacci Sequence with Prime Numbers  
4. Custom Number Sequence Generation  
5. Alternate Signed Sequence Pattern  
6. Factorial of a Number  
7. Mixed Multiplication Sequence Pattern  
8. GCD of Two Numbers  
9. Sum of Harmonic Series (1/1 + 1/2 + 1/3 + ...)  
10. Display Alphabets (A–Z) using Loop  
11. Power of a Number  
12. Factors of a Number  
13. Number as Sum of Two Prime Numbers  
14. Palindrome Number Check  
15. Random Number Generation  
16. Sum of Numbers Divisible by 7 (100–200)

## Concepts Used

- Loops (for, while)
- Conditional Statements
- Number Logic
- Prime Number Algorithms
- Pattern Generation
- Mathematical Computation
- User Input using Scanner
- Object-Oriented Programming Basics

## Language & Tools

- Language: Java
- IDE: VS Code
- Concepts: OOP, Loops, Number Problems
