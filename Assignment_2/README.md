# Java Assignment II – Arrays & Strings

MCA Object Oriented Programming with Java Assignment

This assignment focuses on Java string manipulation, recursion, arrays, text processing, and practical problem-solving programs.

## Topics Covered

1. Longest Common Prefix  
2. Password Validation System  
3. Letter Combinations of a Phone Number  
4. String Permutations using Recursion  
5. Anagram Checker  
6. Profanity Filter for Text  
7. Unique Character Detection  
8. Longest Word Finder  
9. Sentence to Title Case Converter  
10. Domain Extraction from URL

## Concepts Used

- Arrays
- String Manipulation
- Recursion
- Character Operations
- Pattern Matching
- Input Validation
- Searching Logic
- Text Processing
- Object-Oriented Programming

## Language & Tools

- Language: Java
- IDE: VS Code
- Concepts: OOP, Arrays, Strings, Recursion
